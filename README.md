📊 Excel Dashboard Projects
Welcome to my Excel Dashboard Projects repository! 
This repository contains a collection of data analysis and visualization projects built entirely in Microsoft Excel, showcasing my skills in data cleaning, analysis, and dashboard design.
Each project focuses on transforming raw data into clear, interactive, and insight-driven dashboards that support data-driven decision-making.

🔍 What You’ll Find in This Repository
📈 Interactive Excel Dashboards
📊 Pivot Tables, Pivot Charts, and Slicers
📐 Well-structured layouts for storytelling and reporting

🛠 Tools & Skills Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers & Timelines
- Data Analysis & Visualization
- Dashboard Design Principles
- Basic Data Modeling in Excel

This repository is intended to:

🚀 How to Use the Files
- Download or clone the repository
- Open the .xlsx files using Microsoft Excel (desktop version recommended)
- Enable editing and content if prompted
- Use slicers and filters to interact with the dashboards
